# Site GitHub Pages

Ce dépôt contient un site statique prêt à être publié sur GitHub Pages.

## Fichiers principaux
- index.html : page d’accueil
- styles.css : styles du site
- .nojekyll : permet d’éviter le traitement Jekyll
- .github/workflows/deploy-pages.yml : déploiement automatique vers GitHub Pages

## Publication
1. Pousser ce dépôt sur GitHub.
2. Dans le dépôt, ouvrir Settings > Pages.
3. Choisir Source : GitHub Actions.
4. Valider le workflow et attendre le déploiement.

La page sera disponible à l’URL suivante :
https://<votre-utilisateur>.github.io/<nom-du-dépôt>/
